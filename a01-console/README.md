Basic console program 

Counts, writes a random number and prints the value at A0, Whether it is connected to anything or not.



[a01-console:](https://youtu.be/chaqZgd_6Vs?list=PL57Dnr1H_egsL0r4RXPA4PY2yZhOJk5Nr&t=5s)  A simple console program to get things started. No webpage needed. outputs A0 to the console. Then uses a GND (black), 3V3 (red) , SIG (yellow) sensor in this case a [slide potentiometer](https://www.seeedstudio.com/Grove-Slide-Potentiometer-p-1196.html) (variable resitor).
